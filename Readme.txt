Put the client.py file on the server/PC you want to control
Once the client.py file is running on the client device, (python3 client.py)
Run your multi_server.py file. (python3 client.py)
It will listen to the client python file, for when it is active. And when it will be, it will connect to it automatically.

Once connected an interactive Prompt will be initiated.

How to use Interactive prompt
# Interactive prompt for sending commands
# Scooby> list  --> To get list of Ip's connected with their port numbers.

# 1 Friend-B Port
